49 
 * </code>

 * </pre>

 *

 * either an API key or a username/password pair. You can define it using:

 *

 * <pre>

 * <code>

59 
 *       .apiKey("foo")
60 
 * </code>
61 
 * </pre>
62 
 *
63 
 * Or you can also use property keys (using interpolation):
64 
 *
65 
 * <pre>
66 
 * <code>

 * msgBuilder

 *       .set("custom.property.for.api-key", "foo")
71 
 *       .and()
72 
 *    .and()

 * .email()

 *       .apiKey("${custom.property.for.api-key}")

 * </code>

 * </pre>
